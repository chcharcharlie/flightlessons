# FlightLessons

A comprehensive flight instruction management system for Certified Flight Instructors (CFIs) and their students.

## Overview

FlightLessons helps CFIs organize their teaching materials, track student progress, and ensure complete coverage of FAA Airman Certification Standards (ACS). Students get clear visibility into their progress and access to study materials.

## Key Features

- **Study Item Management**: Hierarchical organization of ground and flight training items
- **ACS Integration**: Complete mapping to Private, Instrument, and Commercial requirements
- **Progress Tracking**: Historical tracking with ground (3-state) and flight (1-5 scale) scoring
- **Lesson Planning**: Reusable templates and actual lesson execution tracking
- **Student Dashboard**: Progress visualization and upcoming lesson preparation
- **Mobile Optimized**: Responsive design for use in the field

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Redux Toolkit
- **Backend**: Node.js, Express, TypeScript, Prisma
- **Database**: PostgreSQL, Redis
- **Infrastructure**: AWS (EC2, RDS, S3), Docker
